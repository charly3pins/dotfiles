# /tasks

Delegate to the tasks sub-agent.

The sub-agent will:
1. Read the proposal, specs, and design
2. Break down the change into concrete tasks
3. Organize tasks by phase (Foundation → Implementation → Testing → Cleanup)
4. Save to `openspec/changes/{change-name}/tasks.md`
